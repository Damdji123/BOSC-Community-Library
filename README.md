# BOSC-Community-Library

A simple library management system for the BOSC Community Library.

## Features

- Add books and members
- Borrow and return books
- List books and members
- Data persistence using JSON files

## How to Run

1. Ensure Python is installed.
2. Navigate to the `src` directory.
3. Run `python main.py`

## Project Structure

- `src/models.py`: Contains the Book, Member, and Library classes
- `src/main.py`: The main interface script
- `data/`: Directory for storing JSON data files
